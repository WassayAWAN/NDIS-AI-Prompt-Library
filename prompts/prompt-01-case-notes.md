# Prompt 1 – Case Note Generator

## Prompt Text
Convert raw support worker notes into a structured NDIS-compliant case note using objective, respectful language. Include a clear timeline and sections: Morning Routine, Activities, Behaviour, Support, Response, Outcome.

## Workflow
Daily case note documentation

## Problem
Inconsistent and rushed notes

## Automation
Standardises and saves time

## Risks
Context misinterpretation, requires human review
